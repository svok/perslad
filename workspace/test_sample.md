# Sample Documentation

## Overview

This is a sample markdown document for testing the initial workspace scan functionality.

## Features

- Feature 1: Basic text processing
- Feature 2: Code analysis
- Feature 3: Document indexing

## Usage

```python
from test_sample import DataProcessor

processor = DataProcessor("example")
processor.add_item({"key": "value"})
```

## Configuration

The system supports the following configuration options:

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `name` | str | "" | Processor name |
| `debug` | bool | False | Enable debug mode |

## API Reference

### `calculate_sum(numbers: List[int]) -> int`

Calculate the sum of a list of integers.

### `greet(name: str) -> str`

Generate a greeting message.
